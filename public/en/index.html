Coding Components of WORDPRESS Themes

A Japanese web agency wanted to create their own WordPress themes. I coded sections of each theme with HTML, CSS, and Javascript (mostly jQuery). No frameworks used. Every component is responsive.

  • Component１

    Component：Global navigation
    Features：An accordion menu for mobile

    • Component２−３

    Component：List of news articles
    Features：Different animations on hover, hexagonal title

    Component４

    Component：List of services
    Features：Nothing in particular. Used CSS Grid for the layout.

    Component５−７

    Component：Global navigation, ast of news articles, footer
    Features：Animation on hover in global navigation, especially for the border bottom of each menu and the arrow of each sub menu.

    Component８−９

    Component：List of recipes, footer
    Features：You can move the slider by grabbing either the scrollbar or the slider itself.

    Component１０

    Component：Global Navigation
    Features：Diagonal dotted lines next to page logo, toggle effect for mobile menu

    Component１１−１３

    Component：List of services, footer
    Features：Custom image slider, a button to scroll back to the top of the page

    Component１４−１６

    Component：Global navigation, office hours table, footer
    Features：Slide-in and slide-out effect of mobile menu, accordion menu for mobile, well-organized layout
